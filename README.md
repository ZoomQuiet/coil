Coil
====

Coil is a minimalistic blog engine written in elixir.

Install
-------

Clone the repo and fetch dependencies:

    git clone https://github.com/badosu/coil
    cd coil && mix deps.get

Usage
-----

Edit `config.yml`.

Add an article: `mix post`

And run:

    PORT=3000 mix run --no-halt
